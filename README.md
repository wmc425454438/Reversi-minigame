# 三国黑白棋（Reversi Mini Game）

微信小游戏，国风题材的黑白棋策略对战。融合武将卡牌技能、势力克制和剧情章节系统。

## 玩法

- **基础规则**：6×6棋盘，夹击翻转棋子，与经典黑白棋一致。
- **武将卡牌**：落子需选择手牌中的武将，卡牌攻击力决定伤害，连击值在翻转时触发额外伤害。
- **闪电动画**：落子夹击两端若均为武将，触发闪电链翻转特效。
- **兵力HP**：每人初始120HP，伤害计算=卡牌攻击力+连击加成+翻转加成，HP归零即败。

## 模式

- **对战模式**：选择魏/蜀/吴势力，自定义卡组，简单/中等/困难/PVP。
- **剧情模式**：三国故事章节推进，对话系统（打字机效果+角色立绘），每章特殊规则，分支选择影响走向。

## 系统

- **卡组管理**：多卡组创建/编辑/删除，阵营筛选，点击编入/移出，滑动浏览，本地持久化。
- **胜利/失败动画**：国风水墨风格全屏动画，金色大胜/暗色兵败，粒子特效，点击跳过。
- **武将系统**：魏蜀吴三方约58名武将，攻/连数值+稀有度（金/紫/蓝）。

## 技术栈

- 微信小游戏 Canvas 2D
- 纯 JavaScript ES6 模块
- 数据持久化 wx.setStorageSync
- 触屏交互（点击/拖拽/长按/滑动）

```
└── js/
    ├── main.js          # 主控制器（场景/触控/AI）
    ├── game/            # 游戏逻辑（棋盘/翻转/常量）
    ├── models/          # 数据模型（角色/卡牌/章节/卡组）
    ├── ui/              # 渲染模块（棋盘/卡牌/菜单/对话/动画等）

```
